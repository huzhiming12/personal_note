# 数据流中的中位数

> 题目：如何得到一个数据流中的中位数？如果从数据流中读出奇数个数值，那么中位数就是所有数值排序之后位于中间的数值。如果从数据流中读出偶数个数值，那么中位数就是所有数值排序之后中间两个数的平均值。

思路：建立一个极大堆和一个极小堆，极大堆存放较大的元素，极小堆存放较小的元素。

```Java
import java.util.*;

public class Solution {
    
    int length=0;
    Queue<Integer> maxHeap = new PriorityQueue<>();
    Queue<Integer> minHeap = new PriorityQueue<>(20,new Comparator<Integer>(){
        public int compare(Integer o1,Integer o2)
        {
            return o2-o1;
        }
    });

    public void Insert(Integer num) 
    {
        if(length==0)
        {
            maxHeap.add(num);
            length++;
            return;
        }
        
        if(length%2==0)
        {
            if(minHeap.peek()<num)
                maxHeap.add(num);
            else
            {
                int temp = minHeap.remove();
                minHeap.add(num);
                maxHeap.add(temp);
            }
        }else
        {
            if(maxHeap.peek()<num)
            {
                int temp = maxHeap.remove();
                maxHeap.add(num);
                minHeap.add(temp);
            }
            else
                minHeap.add(num);
        }
        length++;
    }

    public Double GetMedian() {
        if(length==0)
            return 0.0;
        if(length%2==0)
            return (minHeap.peek()+maxHeap.peek())*1.0/2;
        else
            return maxHeap.peek()*1.0;
    }


}
```
