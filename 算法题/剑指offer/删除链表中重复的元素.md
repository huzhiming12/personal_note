# 删除链表中重复的元素

> 题目：在一个排序的链表中，存在重复的结点，请删除该链表中重复的结点，重复的结点不保留，返回链表头指针。 例如，链表1->2->3->3->4->4->5 处理后为 1->2->5

指针p连接的是没有重复的元素，p1指向当前要判断的元素，p2指向

```Java
public class Solution {
    public ListNode deleteDuplication(ListNode pHead)
    {
        if(pHead==null||pHead.next==null)
            return pHead;
        ListNode p = null;
        ListNode p1 = pHead;
        ListNode p2 = pHead.next;
        pHead = null;
        int repeat =1;
        while(p2!=null)
        {
            if(p1.val!=p2.val)
            {
                //表示p1到p2之间没有重复元素
                if(repeat==1)
                {
                   	if(p!=null)
                    	p.next=p1;
                    p=p1;
                    p1=p2;
                    if(pHead==null)
                        pHead=p;
                }else
                {
                    p1=p2;
                }
                repeat=0;
            }
            repeat++;
            p2=p2.next;
        }
        
        if(repeat==1)
        {
            if(p!=null)
            	p.next=p1;
            else
                return p1;
        }else if(p!=null)
            p.next=null;
        
        return pHead;
    }
}
```
