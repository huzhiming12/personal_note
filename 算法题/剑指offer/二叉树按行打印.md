# 二叉树按行打印

题目：

> 从上到下按层打印二叉树，同一层结点从左至右输出。每一层输出一行。

思路：定义两个数current、next，current记录当前行还未访问的结点数，next记录下一行需要访问的结点数。访问当前行结点时，如果左子树不为空则将左孩子加入队列next+1，如果右孩子不为空则将有孩子加入队列next+1，访问完该节点时current—。当current==0时，表示当前行已经访问完

```Java
import java.util.*;
/*
public class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;

    public TreeNode(int val) {
        this.val = val;

}

public class Solution {
    ArrayList<ArrayList<Integer> > Print(TreeNode pRoot) {
        ArrayList<ArrayList<Integer> > res = new ArrayList<ArrayList<Integer> >();
        if(pRoot==null)
            return res;
        LinkedList<TreeNode> queue = new LinkedList<>();
        queue.add(pRoot);
        int current=1; //记录当前行结点数
        int next=1;    //记录下一行的结点数
        while(!queue.isEmpty()){
        	current=next;
            next=0;
            ArrayList<Integer>list = new ArrayList<>();
            while(current>0){
                TreeNode node = queue.remove();
                list.add(node.val);
                if(node.left!=null)
                {   //当前结点的左子树不为空，则将左孩子加入队列，next加1
                    queue.add(node.left);
                    next++;
                }
                if(node.right!=null)
                {	//当前结点的右子树不为空，则将右孩子加入队列，next加1
                    queue.add(node.right);
                    next++;
                }
                current--;  //当前行未访问的结点数减1
            }
            res.add(list);
        }
        return res;
    }
}
```
